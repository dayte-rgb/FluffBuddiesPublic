## Project Description
Paw Patrol is a platform where pets can both seek and offer help to other pets. This is accomplished through the main functionality of the system, 
creating and booking jobs! Pets seeking help can create a listing, which other pets can then book in order to assist. Pets can also communicate with 
each other using real-time messaging, as well as earn badges or compete for the top spots on public leaderboards.

## Features
In this program, users can:
- Create accounts and log in
- Search for jobs, aided by different filtering options
- Create job listings
- Book jobs
- Leave reviews on completed job listings
- View their schedule of created and booked listings
- Message other users
- View their ranks on leaderboards
- Create and update a profile page
- Earn badges

## Installation Instructions
All necessary packages should already be inside the repository. In the event some packages are missing, run ```npm install``` in the terminal, 
which should automatically install all necessary packages.

## How to Start the Program
In the terminal, run ```npm start``` to begin running the program at ```localhost:3000```. Then open any browser and put ```localhost:3000``` in 
the address bar to start using the site.

## How to run tests
In the terminal, run ```npm test```, which should automatically run all test files using Mocha.
